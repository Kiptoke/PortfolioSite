I've made a bunch of programs and other miscellaneous things! Here they are:

PyPIR BotDecember 2019

A bot for Discord that allows server users to play the price guessing game from "The Price is Right" using prices scraped from Amazon product links.

Written using Python 3.8, discord.py, requests, and BeautifulSoup 4

A series of small Python programs that I made to improve and expand my knowledge of the Python programming language. Included are a caesar cipher, calculator, and three games: hangman, guess the number, and InfiniDungeon, a basic dungeon crawler.
