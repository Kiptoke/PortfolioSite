Project 1 - Legend of Zelda RemakeSeptember 2020

A remake of the NES classic The Legend of Zelda in Unity.

All of the features were implemented in a 2 person team using C#, we also included a new dungeon with a new feature, this can be accessed by pressing 4.

Arrow keys for movement, X-key for A, Z-key for B, change items using F1-F4. Press 4 to access the new content.
